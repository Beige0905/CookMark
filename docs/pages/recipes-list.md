# `/recipes` — 레시피 보관함

## 역할
저장된 레시피 전체 목록을 그리드로 표시. 검색·정렬 UI가 있으나 현재 더미 데이터(6건 플레이스홀더)로 표시되며 API 미연동.

## 사용자 액션

| 액션 | 트리거 | API 호출 | 성공 | 실패 |
|------|--------|----------|------|------|
| + 새 레시피 추가 버튼 | `click` (a 태그) | 없음 | `/recipes/new` 로 이동 | — |
| 검색 인풋 | `input` | 없음 | UI만 (필터링 미구현) | — |
| 정렬 select | `change` | 없음 | UI만 (정렬 미구현) | — |
| 레시피 카드 클릭 | `click` | 없음 | 미구현 (링크 없음) | — |

## 에러 케이스
없음 — API 호출 없음.

## 미구현 항목
- 실제 레시피 목록 → `GET /api/recipes` 연동 필요
- 검색 → 클라이언트 필터 또는 서버 검색 API 필요
- 정렬 → 클라이언트 정렬 또는 서버 파라미터 필요
- 레시피 카드 → `/recipes/{id}` 링크 연결 필요
