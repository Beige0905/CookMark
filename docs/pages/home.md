# `/` — 홈 대시보드

## 역할
앱 진입 화면. 오늘의 추천 레시피, 내 레시피 보관함 미리보기(3건), 링크 입력 바를 보여준다.
현재 표시 데이터는 하드코딩된 더미 데이터이며 API 연동 없음.

## 사용자 액션

| 액션 | 트리거 | API 호출 | 성공 | 실패 |
|------|--------|----------|------|------|
| 링크 입력 후 Enter | `keydown Enter` | 없음 | `alert('링크 저장 예정: ...')` (미구현) | — |
| 저장 버튼 클릭 | `click` | 없음 | `alert('링크 저장 예정: ...')` (미구현) | — |
| 레시피 카드 클릭 | `click` / `Enter` | 없음 | `/recipes/{id}` 로 이동 | — |
| 전체 보기 → 버튼 | `click` | 없음 | 미구현 (이벤트 핸들러 없음) | — |
| 만들었어! 🔥 버튼 | `click` | 없음 | `alert('만들었어!')` (미구현) | — |

## 에러 케이스
없음 — API 호출이 없어 에러 상태 없음.

## 상태
- `linkInput` — URL 입력값 (빈 문자열이면 저장 버튼 미노출)

## 연관 컴포넌트
- `+layout.svelte` (사이드바/네비게이션)

## 미구현 항목
- 링크 저장 → 실제 API 연동 필요
- 더미 레시피 → `GET /api/recipes` 연동 필요
- 오늘의 추천 → 추천 로직 미정
- "만들었어!" → 요리 기록 기능 미정
