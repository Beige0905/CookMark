# `/fridge` — 내 냉장고

## 역할
냉장고 재료를 관리하고, 보유 재료 기반 레시피 추천을 받는 페이지.
현재 UI 셸만 구현된 상태이며 모든 기능이 미구현.

## 사용자 액션

| 액션 | 트리거 | API 호출 | 성공 | 실패 |
|------|--------|----------|------|------|
| + 재료 추가 버튼 | `click` | 없음 | 미구현 | — |
| 재료 × 버튼 | `click` | 없음 | 미구현 | — |
| 🍳 이 재료들로 뭐 만들지? | `click` | 없음 | 미구현 | — |
| 유통기한 임박 우선 체크박스 | `change` | 없음 | UI 토글만 (기능 없음) | — |
| 메인 재료 포함 체크박스 | `change` | 없음 | UI 토글만 (기능 없음) | — |

## 에러 케이스
없음 — API 호출 없음.

## 미구현 항목
- 재료 CRUD → API 미정
- 레시피 추천 → AI 연동 미정
- 필터 → 로직 미정
