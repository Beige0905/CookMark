# `/settings` — 설정

## 역할
계정 정보 확인 및 앱 환경 설정 페이지. 현재 UI 셸만 존재하며 모든 버튼/토글이 미구현.

## 사용자 액션

| 액션 | 트리거 | API 호출 | 성공 | 실패 |
|------|--------|----------|------|------|
| 계정 수정 버튼 | `click` | 없음 | 미구현 | — |
| 로그아웃 버튼 | `click` | 없음 | 미구현 | — |
| 다크 모드 토글 | `click` | 없음 | 미구현 (토글 상태 변경 없음) | — |
| 푸시 알림 토글 | `click` | 없음 | 미구현 (토글 상태 변경 없음) | — |

## 에러 케이스
없음 — API 호출 없음.

## 표시 정보
- 사용자 이름: 하드코딩 "사용자 님"
- 이메일: 하드코딩 "user@example.com"
- 앱 버전: 하드코딩 "1.0.0 (beta)"

## 미구현 항목
- 계정 정보 수정 → API 미정
- 로그아웃 → 인증 시스템 미정
- 다크 모드 → 테마 시스템 미정
- 푸시 알림 → 알림 시스템 미정
- 실제 사용자 정보 로드 → 인증 API 미정
