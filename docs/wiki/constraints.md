### Claude Code Stop 훅 — 대화 내용 접근 불가

Stop 훅은 셸 커맨드라서 현재 세션의 대화 내용에 직접 접근할 수 없음. 외부 스크립트에서 Claude API를 호출해도 트랜스크립트를 넘기는 방법이 복잡함. → 위키 자동 업데이트는 CLAUDE.md 지시로 Claude가 세션 중 직접 파일을 쓰는 방식으로 대체.
